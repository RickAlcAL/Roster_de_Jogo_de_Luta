# Roster_de_Jogo_de_Luta
Aplicação Java voltada para a classificação e gerenciamento de personagens de jogos (RPG). O sistema processa dados extraídos de um arquivo CSV estruturado, separa os registros dinamicamente entre as subclasses especializadas Atirador e Lutador Corpo a Corpo utilizando herança e polimorfismo, e exibe as estatísticas formatadas no console.
